# Daten

Dieses Verzeichnis ist für Rohdaten und statistische Grundlagen reserviert,
z. B. offizielle THG-Emissionsdaten, Energieverbrauchsstatistiken und
Bevölkerungszahlen, die in den Modellrechnungen verwendet werden.

## Mögliche Inhalte

- Treibhausgasemissionen nach Sektoren (Umweltbundesamt)
- Energieverbrauch und Erzeugung (AGEB, Bundesnetzagentur)
- Ackerflächen und Leguminosen-Potenziale (Statistisches Bundesamt / BMEL)
- ETS-Zertifikatspreise und CO₂-Bepreisung (EEX, DEHSt)
